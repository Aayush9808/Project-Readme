## 🎓 **Technical Deep Dive**

### **Core Algorithms**

#### **1. Ambulance Allocation Algorithm**

**Objective**: Find the optimal ambulance for an emergency request

**Process**:
1. Filter all ambulances with "AVAILABLE" status
2. For each available ambulance, calculate ETA using shortest path
3. Consider traffic multiplier (1.0 - 3.0 based on congestion)
4. Select ambulance with minimum ETA
5. Assign and update status

**ETA Formula**:
$$
\text{ETA} = \frac{\text{ShortestPathDistance}}{\text{AverageSpeed}} \times \text{TrafficMultiplier}
$$

**Time Complexity**: $O(A \times E \log V)$ where A = ambulances, E = edges, V = vertices

---

#### **2. Shortest Path Routing**

**Dijkstra's Algorithm**:
- Guaranteed optimal path
- Priority queue implementation
- Time Complexity: $O(E \log V)$

**A* Optimization**:
- Heuristic-based search
- Reduces nodes explored by 40-60%
- Heuristic Function: $f(n) = g(n) + h(n)$

---

#### **3. Machine Learning Prediction**

**Problem**: Predict emergency demand by location and time

**Features**:
- Temporal: Hour, day, month, holidays
- Spatial: Latitude, longitude, population density
- Historical: Past emergency counts
- Environmental: Weather, temperature, events

**Models**:
- Random Forest (Classification)
- XGBoost (Regression)
- LSTM (Time-Series)

**Performance**: 82-88% accuracy

## 🛠️ **Installation & Setup Guide**

### **Prerequisites**

- Node.js (v18+)
- Python (v3.9+)
- PostgreSQL (v14+) with PostGIS
- Redis (v7+)
- Docker & Docker Compose (optional)

### **Quick Start with Docker**

```bash
# Clone repository
git clone https://github.com/yourusername/mediroutex.git
cd mediroutex

# Start all services
docker-compose up -d

# Access application
# Frontend: http://localhost:3000
# Backend API: http://localhost:3000/api
# ML Service: http://localhost:5000
```

### **Manual Setup**

**Backend:**
```bash
cd backend
npm install
cp .env.example .env
npm run migrate
npm run seed
npm run dev
```

**ML Service:**
```bash
cd ml-service
python -m venv venv
source venv/bin/activate
pip install -r requirements.txt
python src/app.py
```

**Frontend:**
```bash
cd frontend
npm install
npm start
```
